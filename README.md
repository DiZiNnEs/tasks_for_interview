# Задача №3

## Принцип работы
С клиенсткой части отправляется POST запрос по следующему адресу 
###
`127.0.0.1:8000/api/multiply/1234`
###
Ответ будет слудеющим
```json
{
    "Вы ввели следующие значения": [
        1,
        2,
        3,
        4
    ],
    "Результат": 24
}
```
